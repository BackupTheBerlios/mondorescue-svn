1.1. Purpose / Scope of this Guide

This HOWTO serves to help people get started with using mondo/mindi as way to backup their system. This is a work constantly evolving. It was was started by Hugo Rabson and has since been worked upon by a number of people. See the 'About' web page for a list of members of the development team.

The newest version of this document can always be found on Mondo's homepage http://www.mondorescue.org.

There are many ways to contribute to the Linux movement without actually writing code. One of the most important is writing documentation, allowing each person to share their knowledge with thousands of others around the world. This HOWTO is designed to help you get familiar with how Mondo/Mindi works.
