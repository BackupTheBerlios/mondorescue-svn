4.3. Linux Kernel Modules

Mondo (specifically Mindi) does not require any specific modules. It does require that your kernel support the initrd initial ramdisk facility. Typically this is supported by the Linux kernel. Modules used are needed to support the CD, floppy disks, hard disks, etc. If the support is modular, then the modules will be incorporated in a boot disk by Mindi. If the support is built-in (static), then it will be available at boot-time by default.
